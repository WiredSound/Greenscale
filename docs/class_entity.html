Detailed Description

Entities exist in instances of GameMap (or more specifically in GameMap owned EntityLayer instances) and are capable of moving and interacting with the game world around them.

The ID of the entity determines the animation, default components, name, description and other such static values held in the relevant EntityInfo struct contained in the EntityManager. Values that vary between entities, such as the current equipped components, aligned factions, position, as well as the behaviour (primarily via the held EntityController pointer) are stored in the Entity class itself.
