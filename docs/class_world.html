Detailed Description

The World class is tasked with creating and managing connected maps that together make up the overworld.

It calls upon the TurnManager to allow for the turn system to work and uses the MapBuilder to construct new maps. It also keeps a list of player controlled/required entities so that these entities can be moved between maps.
