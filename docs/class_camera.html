Detailed Description

Handles the movement of the view around the game world.

The Camera class is tasked with handling changes to the render window's sf::View instance in order to move the camera around the world space. It also handles smoothly moving the view to a target point in space (via the moveTowardsCentre method) as well as ensuring the GUI remains static. It has update method which should be called every frame.
