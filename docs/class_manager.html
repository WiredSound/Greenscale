Detailed Description

template<typename Key, typename Managed>
class Manager< Key, Managed >

Manager is a class template that is designed to make the loading of tiles, projectiles, entities, components and component upgrades from JSON files easier.

Template Parameters
KeyThe key that points to a managed item. Typically an enum stored in the IDs namespace.
ManagedAn object/struct storing all the data relevant to what is being managed.
